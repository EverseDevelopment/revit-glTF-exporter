## Getting Started

1. Go to a 3D View

2. Hide the elements you dont want to exporter

3. Select the properties to export

4. Hit export button

 
You are done! 

## Current Version
glTF exporter is currently built against Revit 2019,2020,2021,2022,2023 and 2024
